## Порядок предоставления почтовых ящиков.[[править](http://10.1.115.172:8080/mediawiki/index.php?title=%D0%A1%D0%97_%D0%BD%D0%B0_%D1%81%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5_%D0%BF%D0%BE%D1%87%D1%82%D0%BE%D0%B2%D1%8B%D1%85_%D1%8F%D1%89%D0%B8%D0%BA%D0%BE%D0%B2&action=edit&section=1 "Редактировать раздел «Порядок предоставления почтовых ящиков.»")]

При обращении пользователей необходимо оказывать консультацию для действий по маршруту указанному ниже.

Для создания почтового ящика необходимо написать СЗ в Директум на Веселову О.В., выбирать маршрут **«Служебная записка на создание электронной почты».**

Маршрут следующий:

1. СА создают почтовый ящик
2. СА делают заявку на почту support2915@pirogov-center.ru с приложенной СЗ, «Прошу настроить новый почтовый ящик сотруднику _____»
3. СТП в заявке выбирают услугу **2.2.2 Создание/настройка почтового ящика**, настраивают почтовый ящик пользователю и закрывают заявку с указанием решения.

  
**При оказании консультации не забываем заводить заявки, в заявках с консультацией о создании почтового ящика выбирайте услугу 2.2.1 Доменная почта.**
