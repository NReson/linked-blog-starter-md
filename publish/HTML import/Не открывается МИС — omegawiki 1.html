Не открывается МИС

Материал из omegawiki

Если при открытии MSClinic возникает ошибка то необходимо выполнить следующие действия.

Обновление групповых политик центра[править]

  • Открываем CMD под пользователем
  • Вводим в командную строку команду gpupdate /force
  • Закрываем командную строку

Установка Яндекс браузера[править]

Установка осуществляется в том случае если Яндекс браузер для организации не установлен на данном АРМ

  1. Если установлен какой либо другой яндекс браузер, то изначально его нужно удалить. Для этого:
    • Переходим в папку \\10.1.31.115\Share\Software\Стандартное ПО (ИБ)\Yandex
    • Запускаем ярлык Удаление обычной версии Яндекс браузера.bat
  2. Для установки переходим в папку \\10.1.31.115\Share\Software\Стандартное ПО (ИБ)\Yandex
  3. Запускаем установщик YandexBrowser.msi
  4. После установки выполняем обновление групповых политик центра gpupdate /force

Изменение Ярлыка MSClinic[править]

Для изменения ярлыка необходимо:

  1. По нужному ярлыку нажать правой кнопкой мыши
  2. После чего выбрать "Свойства"
  3. В поле URL указать актуальный адрес MSClinic (можно посмотреть в Yandex браузере).
