Установку проводить в соответствии со следующим регламентом:

1. Создаем заявку в Инфраменеджере на переустановку ОС на АРМ (услуга **1. АРМ \ 1.3 Техническая и справочная поддержка \ 1.3.9 Переустановка ОС**).

2. Сохраняем файлы пользователя и смотрим, какое ПО было установлено (при необходимости)

3. Устанавливаем ОС Windows с USB-носителя.

4. Создаем локального администратора со следующими данными:

**Логин: support**

**Пароль: 3wvi/3OQ0aGo**

5. Ввод в домен **(при необходимости нужно удалить старое имя в active directory)**, для этого заходим в свойства Компьютера, находим пункт **Переименовать этот ПК (для опытных пользователей)**,

Принцип наименование АРМ: 70-46N000215
70 –двузначное число, номер дома. Для стационара 70, для КДЦ 65, для Арбата 37
46N000215 – Инвентарный номер оборудования. 

[![](./Установка%20ОС%20—%20omegawiki_files/PC1.png)](http://10.1.115.172:8080/mediawiki/index.php/%D0%A4%D0%B0%D0%B9%D0%BB:PC1.png)

далее нажимаем на кнопку **Изменить**,

[![](./Установка%20ОС%20—%20omegawiki_files/PC2.png)](http://10.1.115.172:8080/mediawiki/index.php/%D0%A4%D0%B0%D0%B9%D0%BB:PC2.png)

вводим правильное доменное имя и сам домен.

[![](./Установка%20ОС%20—%20omegawiki_files/PC3.png)](http://10.1.115.172:8080/mediawiki/index.php/%D0%A4%D0%B0%D0%B9%D0%BB:PC3.png)

Переводим в нужную OU

6. Перевод заявки на ИБ, с указанием названия подразделения, корпуса, этажа и кабинета, в котором будет стоять данный АРМ. ИБ производит установку Secret Net Studio и Kaspersky (Если заявки на переустановку не было, то отдельную заявку на установку SNS и Kaspersky).

7. ИБ переводит заявку обратно на СТП.

8. Переносим файлы пользователей (при необходимости).

9. Устанавливаем доп. программ по потребностям пользователя (МИС+Парус и т.д., при необходимости).
